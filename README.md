# Flipkart Customer Satisfaction (CSAT) Prediction Using Machine Learning

## Project Overview

This project focuses on predicting customer satisfaction (CSAT) scores for Flipkart support tickets using Machine Learning techniques. The main objective is to analyze customer support data, perform Exploratory Data Analysis (EDA), preprocess the dataset, and build classification models capable of predicting customer satisfaction levels.

The project demonstrates the complete Machine Learning workflow including:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Data Preprocessing
* Model Building
* Model Evaluation
* Performance Comparison

---

## Problem Statement

Customer satisfaction is one of the most important factors for any e-commerce company. Flipkart receives thousands of customer support tickets related to orders, payments, delivery, returns, refunds, and product issues.

The goal of this project is to predict whether a customer is satisfied or dissatisfied based on ticket-related information using Machine Learning classification algorithms.

---

## Dataset Information

The dataset contains customer support ticket information such as:

* Support Channel
* Ticket Category
* Issue Type
* Response Time

* Customer Interaction Details
* CSAT Score (Target Variable)
